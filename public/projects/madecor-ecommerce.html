Sobre el Proyecto

Plataforma de ecommerce para la venta de material didáctico, con catálogo de productos, carrito de compras, pasarela de pago y generación de facturas. Desarrollada con Spring Boot en el backend y Angular en el frontend.

Stack de desarrollo: El backend fue desarrollado con Spring Boot (Java), proporcionando una arquitectura empresarial robusta y escalable. El frontend utiliza Angular para crear una experiencia de usuario moderna e interactiva. La base de datos PostgreSQL garantiza la integridad y rendimiento de las transacciones comerciales.

Tecnologías implementadas: Stripe fue integrado como pasarela de pago, proporcionando procesamiento seguro de transacciones y múltiples métodos de pago para los clientes. El sistema incluye funcionalidades completas de e-commerce con gestión de inventario y generación automática de facturas.

Características Implementadas

  • ✅ Plataforma de e-commerce completa
  • ✅ Gestión de productos de decoración
  • ✅ Integración de pagos con Stripe
  • ✅ Sistema de inventario automatizado
