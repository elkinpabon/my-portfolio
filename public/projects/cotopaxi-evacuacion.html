App Cotopaxi

Sistema de evacuación de emergencias volcánicas

Sobre el Proyecto

App Cotopaxi es una aplicación móvil desarrollada para el Instituto Geofísico del Ecuador como sistema de alerta y evacuación ante posibles erupciones del volcán Cotopaxi. Incluye geolocalización GPS, rutas de evacuación optimizadas, alertas push en tiempo real, mapas offline y coordinación con centros de emergencia. Desarrollado en 6 meses para 50,000+ habitantes de zonas de riesgo.

Funcionalidades Críticas

  • Alertas push en tiempo real
  • Geolocalización GPS precisa
  • Rutas de evacuación optimizadas
  • Mapas offline para emergencias
  • Coordinación con centros de emergencia
  • Kit de emergencia integrado
  • Contactos de emergencia rápidos
  • Historial de actividad volcánica

Tecnologías utilizadas: React Native para desarrollo multiplataforma, Google Maps API para navegación y rutas, Firebase para notificaciones push, Socket.io para comunicación en tiempo real con centros de comando, y SQLite para almacenamiento offline. La aplicación funciona sin conexión a internet usando mapas y datos pre-descargados, garantizando operatividad durante emergencias.

Impacto Social

Esta aplicación forma parte del plan nacional de gestión de riesgos volcánicos del Ecuador. Permite reducir significativamente los tiempos de evacuación y coordinar eficientemente los recursos de emergencia. Ha sido adoptada por autoridades locales y servicios de emergencia como herramienta oficial para la preparación ante erupciones volcánicas.
