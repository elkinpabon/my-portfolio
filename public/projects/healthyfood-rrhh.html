Sobre el Proyecto

Sistema completo de gestión de recursos humanos para la empresa Healthy Food, diseñado para automatizar procesos como el cálculo de horas trabajadas, la generación de reportes de pago, la gestión de empleados y áreas de trabajo, y la emisión de credenciales con código QR.

Stack de desarrollo: El backend fue desarrollado con Node.js y Express, proporcionando una base sólida y escalable para el manejo de datos. El frontend utiliza React para crear una interfaz de usuario moderna e intuitiva. La base de datos MongoDB permite un almacenamiento flexible y eficiente de la información de empleados y procesos de recursos humanos.

Tecnologías implementadas: Se integró JWT para la autenticación segura del sistema, garantizando que solo usuarios autorizados puedan acceder a la información sensible. QR Code Generator fue implementado para la generación automática de credenciales digitales únicas para cada empleado.

Características Implementadas

  • ✅ Gestión completa de recursos humanos
  • ✅ Sistema de control de asistencias
  • ✅ Generación de credenciales con QR
  • ✅ Autenticación segura con JWT
  • ✅ Interface moderna con React

Equipo de desarrollo: Proyecto desarrollado por 3 personas especializadas en backend, frontend y sistemas.

Tiempo de desarrollo: El proyecto fue completado en un período de 3 meses, incluyendo análisis de requerimientos, desarrollo, testing y deployment del sistema completo.
