Sobre el Proyecto

Aplicación móvil diseñada para el control de asistencia de empleados en tiempo real. Desarrollada para tablets, permite registrar la entrada y salida mediante escaneo de QR o PIN. Está conectada a un servicio web que registra cualquier novedad y envía notificaciones a los administradores.

Stack de desarrollo: El frontend fue desarrollado con Flutter y Dart, proporcionando una experiencia nativa multiplataforma optimizada para tablets. El backend utiliza Node.js con Express para manejar las operaciones del servidor de manera eficiente. La base de datos MongoDB almacena toda la información de asistencias y empleados de forma segura y escalable.

Tecnologías implementadas: Firebase fue integrado para proporcionar notificaciones en tiempo real, asegurando que los administradores reciban alertas instantáneas sobre cualquier novedad en el sistema de asistencias. El sistema de autenticación mediante QR y PIN garantiza un acceso rápido y seguro.

Características Implementadas

  • ✅ Interfaz móvil optimizada para tablets
  • ✅ Aplicación multiplataforma con Flutter
  • ✅ Notificaciones push con Firebase
  • ✅ Backend robusto con Node.js
  • ✅ Base de datos escalable con MongoDB
