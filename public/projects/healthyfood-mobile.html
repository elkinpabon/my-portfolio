Aplicación Móvil Control de Asistencias

HealthyFood - App nativa para gestión de asistencias

Cliente

HealthyFood S.A.

Aplicación complementaria al sistema web de RRHH para optimizar el control de asistencias en tiempo real.

Tiempo de Desarrollo

4 meses

Desarrollo nativo para Android e iOS, incluyendo integración con sistemas biométricos.

Equipo de Trabajo

4 profesionales

  • 1 Project Manager
  • 2 Desarrolladores Mobile
  • 1 UI/UX Designer especializado en móvil

Tecnologías Utilizadas

Flutter Dart Biometric SDK SQLite REST API Push Notifications

Descripción del Proyecto

Desarrollamos una aplicación móvil nativa multiplataforma que permite a los empleados de HealthyFood registrar su asistencia de manera rápida y segura mediante autenticación biométrica.


Funcionalidades principales:

  • Autenticación Biométrica: Huella dactilar y reconocimiento facial
  • Check-in/Check-out: Registro de entrada y salida con geolocalización
  • Modo Offline: Funcionamiento sin conexión con sincronización automática
  • Historial Personal: Consulta de asistencias y horas trabajadas
  • Solicitud de Permisos: Gestión digital de ausencias y permisos
  • Notificaciones Push: Recordatorios y comunicados importantes
  • Dashboard Personal: Métricas individuales y estado actual
  • Modo Supervisor: Vista especial para coordinadores y jefes

La aplicación incrementó la precisión del control de asistencias en un 95% y redujo el tiempo de registro a menos de 3 segundos por empleado, mejorando significativamente la experiencia del usuario y la eficiencia operacional.
