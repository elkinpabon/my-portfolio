# Google Search Console - Verificación del sitio
# 
# Para verificar tu sitio en Google Search Console:
# 1. Ve a: https://search.google.com/search-console
# 2. Agrega tu propiedad: my-portfolio-henna-eta-55.vercel.app
# 3. Selecciona método de verificación "Etiqueta HTML"
# 4. Copia el código meta y agrégalo en el <head> de index.html
#
# Ejemplo:
# <meta name="google-site-verification" content="TU_CODIGO_AQUI" />
#
# Después de verificar, podrás:
# - Enviar el sitemap.xml
# - Ver estadísticas de búsqueda
# - Monitorear el rendimiento SEO
# - Detectar errores de indexación
